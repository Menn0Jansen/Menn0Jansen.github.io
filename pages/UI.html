User Interface

UI (user interface) wordt gebruikt om mensen te laten communiceren met een computer. Het is dus de manier waarop mensen met de toepassingelaag interacteren. Dit geldt voor zowel input als output. Hier zijn enkele voorbeelden: knoppen, navigatiebalken, touchscreen, speaker, microfoon, schermen, etc. In het hierop volgende deel wordt vooral ingegaan op de UI van apps en websites.

Principes UI

Om ervoor te zorgen dat UI zijn functie goed voldoet, zijn er bepaalde principes waar bij UI aan voldaan wordt.

  • Duidelijkheid: het moet duidelijk zijn voor de gebruiker hoe hij met de machine kan communiceren.
  • Consistentie: alle elementen moet vergelijkbaar eruit zien zodat het makkelijk is voor de gebruiker om te begrijpen dat ze een verband met elkaar hebben.
  • Uiterlijk: voor de gebruiker moet de UI er ook aantrekkelijk uitzien. Dit zorgt ervoor dat hij meer geneigd is om met de UI te interacteren en dat zal de UX (user experience) ook bevorderen.
  • Efficiëntie: UI moet er voor zorgen dat de gebruiker krijgt wat hij wilt en dit moet ook nog eens efficiënt. Door middel van shortcuts en directe verwijzingen naar belangrijkere functies neemt de efficiëntie toe.

Design van UI

Het design van een werbsite (het uiterlijk) is belangrijk en heeft ook principes.

  • Kleur: kleur is een belangrijk middel dat apps gebruiken om gebruikers te verleiden om hun product te gebruiken. Er moet een balans gevonden worden tussen felle en donkerdere kleuren. Als de kleuren te opvallend zijn, zou dit voor de gebruiker nadelig kunnen zijn. Ook minder opvallende kleuren zouden een negatief effect kunnen hebben: het motiveert de gebruiker niet om op een app-icoon te klikken bijvoorbeeld.
  • Letters: het lettertype moet leesbaar en duidelijk zijn, maar ook aantrekkelijk. Daarnaast moet het groot genoeg zijn voor iedereen om het te kunnen lezen, maar ook niet te groot dat er onvoldoende informatie op het scherm past.
  • Interactie: het toevoegen van interactie zorgt ervoor dat de UX wordt bevordert. Het zorgt ervoor dat de gebruiker actief betrokken is.
  • Layout: de layout van bijvoorbeeld een website moet overzichtelijk zijn; er kan niet te veel tekst op staan, dus wordt er vaak gebruik gemaakt van witregels en gescheidde vakken met andere informatie.
  • Inclusiviteit: het is van belang om de UI voor zo veel mogelijk mensen toegankelijk te maken. Daarom wordt er voor gezorgd dat de UI rekening houdt met mensen met bijvoorbeeld motorische beperkingen of mensen met bijvoorbeeld kleurenblindheid.
  • Trends: op het internet veranderen trends er vaak heel snel, er moet dus rekening gehouden worden met wat er op het moment trending is. Voorbeelden van trends in UI zijn minimalistisch design en dark mode.

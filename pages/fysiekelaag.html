3-Lagenmodel

Een computer opereert door middel van het 3-lagen model. Deze drie lagen (fysieke, logische en toepassingen laag) werken samen om ervoor te zorgen dat de computer kan functioneren. De fysieke laag bestaat uit hardware (scherm, muis, toetsenbord, etc). De volgende laag is de logische laag (databases, internet, netwerken, etc). En de laatste laag is de toepassingslaag (games, browsers, apps, etc). Processor

Een processor (CPU) is hardware in een computer die ervoor zorgt dat programmacode wordt uitgevoerd.

In de ophaalfase neemt de processor de code van een geheugenadres, dit is de code die als eerst zal worden uitgevoerd. Daarna komt de processor in de decodeerfase: de 0's en 1's worden omgezet in instructies. Daarna komt de processor in de uitvoerfase, dan wordt de instructie uitgevoerd. Hierop volgt de opslagfase, de uitkomst van de instructie wordt opgeslagen.

Geheugen

Het geheugen van een computer bewaart informatie, zodat het later weer gebruikt kan worden.

RAM

RAM staat voor random-access-memory. Het is een korte-termijn geheugen dat snel en kort dingen onthoudt en ervoor zorgt dat de CPU het kan uitvoeren.

ROM

In tegenstelling tot RAM, behoudt ROM (Read-only-memory) zijn gegevens ook nadat het is uitgezet.

Permanent

Permanent geheugen, ook wel niet-vluchtig geheugen genoemd, is permanent geheugen waarvan de informatie nog op de computer blijft zelfs nadat de computer is uitgeschakeld. Het is dus langetermijngeheugen. Rom valt onder permanent geheugen, RAM niet.

Randapparatuur

De term 'randapparatuur' valt onder alle onderdelen van niet tot de centrale rekeneenheid behoren. Randapparatuur zorgt ervoor dat de communicatie tussen gebruiker en computer, maar ook tussen verschillende onderdelen van een computer gemakkelijker en efficienter gaat. Voorbeelden van randapparatuur zijn: computermuis, toetsenbord, scanner, videokaart, monitor, modem, harde schijf, USB-stick. Sommige van deze onderdelen lijken onmisbaar in een computer, maar in werkelijkheid heeft een computer alleen geheugen en een CPU nodig om als turgingmachine te voldoen (een machine die berekingen kan uitvoeren).

Kloksnelheid

De kloksnelheid geeft aan hoe vaak per seconde een signaal gelezen, geschreven of verwerkt wordt. Een kloksnelheid van 1 Hz komt overeen met 1 bewerking per seconde. Tegenwoordig werken processors met een kloksnelheid van 3 GHz, dit zijn dus 3 miljard bewerkingen per seconde.

Verwerkingssnelheid

De verwerkingssnelheid werkt met Instructies per seconde (IPS). 1 IPS betekent dat er 1 instructie wordt uitgevoerd per seconde.
